# 🎬 DishFlix

**DishFlix** is a **Reel-style video feed platform** that allows create, upload, and explore short videos with a smooth and responsive interface.

## 🚀 Features
- 🎥 Create and share **Reel-style short videos**  
- 🧑‍💻 User authentication and profile system  
- 💬 Interactive and **responsive video feed**  
- ⚙️ Built with a **modern MERN stack** for performance and scalability  

## 🧩 Tech Stack
**Frontend:** React.js  
**Backend:** Node.js, Express.js  
**Database:** MongoDB
